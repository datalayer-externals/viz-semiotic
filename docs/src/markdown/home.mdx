# Semiotic

**Welcome to Semiotic's documentation for the upcoming 2.0 release.**

Looking for 1.x release documentation? Please visit https://semioticv1.nteract.io.

## Semiotic is a data visualization framework for React.

Semiotic offers frame types, such as XYFrame, OrdinalFrame, and NetworkFrame, to
deploy a wide variety of charts:

- `XYFrame`: XY data i.e. line charts and scatterplots
- `OrdinalFrame`: categorical data i.e. bar charts, violin plots, parallel
  coordinates
- `NetworkFrame`: topological and network data i.e. flow diagrams, network
  visualization, and hierarchical views

## Getting Started

### Installation

Install and save `semiotic` to your project with `yarn` or `npm`.

```bash
yarn add -E semiotic
```

or

```bash
npm i -SE semiotic
```

### Usage

Use with the following import syntax:

```js
import { NetworkFrame } from "semiotic"
```

Alternatively, you can use semiotic's bundled version available on unpkg.com.

```html
<script type="module">
  import { NetworkFrame } from "https://unpkg.com/semiotic?module"
</script>
```
### Explore

Explore the interactive examples and guides to learn more about the flexibility
and power of Semiotic. Along with the semiotic and React components, these
examples rely on some basic
[CSS](https://github.com/nteract/semiotic/blob/main/docs/public/semiotic.css)
for styling.
